== h5util ==
By Fredrik Orderud, 2009.

h5util is a collection of small utility functions to simplify storage and loading of scalar, vectors and tables of any dimensionality and type through the HDF5 C++ interface.

== Dependencies ==
HDF5 C++ library:
Environment variable: RCTL_HDF

Boost C++ libraries (optional):
Environment variable: RCTL_BOOST
